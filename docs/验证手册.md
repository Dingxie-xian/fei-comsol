## 1. 通用检查流程 (在 COMSOL Desktop)

### 1.1 打开模型
1. 双击 `.mph` → COMSOL Multiphysics 6.4 启动
2. 等几秒 — 模型树加载完成(左侧"模型构建器"展开 Component 1)
3. 顶部"模型类型"应该显示 `2D Axisymmetric` (2D 轴对称)

### 1.2 看几何 (Component 1 → Geometry 1 → geom1)
**期望看到**:
- 一个大矩形(r: 0→150, z: -300→+330,Z_down 不同)
- 顶部可能多一个 20mm PML 矩形 + 右侧 20mm PML 矩形 + 角落 20×20 PML(PML 模型)
- 中间 z=0 附近 10 个细环(每个 0.1mm 厚)
- **关键检查点**: 点开 Geometry → 展开 air / zp_1..zp_10 / pml_top / pml_right / pml_corner → 应该有 13-15 个 Boolean 操作

**怎么测**:
- 右键 Geometry 1 → `Plot Geometry` → 应该看到上面的图
- 点击 `Measure` 工具 → 选 Rectangle air → 显示 size 应该 = `{150, 610}` (PML 模型) 或 `{150, 500}` (plane baseline)
- 点 z=0 处的 10 个细环 → 选 Rectangle zp_1 → size 应该 = `{32.22, 0.1}`

### 1.3 看物理 (Component 1 → Pressure Acoustics 1)
**期望看到**(PML 模型 step1_v2_pml_mesh_converged 为例):
- `acpr`: Pressure Acoustics, Frequency Domain
- `src`: Normal Velocity on id=选中, nvel=v_n=2.5 mm/s
- `baffle`: Sound Hard (剩余底部)
- `pmlt_sh / pmlr_sh / pmlc_sh`: Sound Hard (PML 外边界)
- `sh_2, sh_4, sh_6, sh_8, sh_10`: Sound Hard on even rings
- (奇数环 zp_1,3,5,7,9 没有显式 BC → 默认 Continuity = 空气 → 空气)

**怎么验证**:
- 点 acpr → "Physics" 标签 → 列出所有 BC
- 点 src → 蓝色高亮应该显示 z = -300 的中央一小段(r ≈ 0-8 mm),不是整个底部
- 点 baffle → 应该高亮 z = -300 的两侧(r ≈ 8-150 mm)
- 点 sh_2 → 应该高亮 z=0 附近的第 2 个环(径向 32.22-45.96 mm, z ∈ [-0.05, 0.05])

### 1.4 看网格 (Mesh 1)
**期望**(PML 模型,lam/8):
- 单元类型: 自由三角形(Free Triangular)
- 最大尺寸 hmax: c_air/f0/8 = 340/40000/8 = **1.0625 mm**
- 最小尺寸 hmin: 0.1 mm
- 网格统计: 域单元数 ~80,000-100,000 (PML 模型),或 ~50,000 (plane baseline)
- **关键**: z = 0 附近(波带片)网格要细(0.1-0.5 mm),因为这里有 0.1mm 厚的矩形

**怎么验证**:
- Mesh 1 → `Plot Mesh` (或 `Refine` 一次看更细)
- 顶部统计栏: "Number of elements: ~XX,XXX"

### 1.5 看研究 (Study 1)
**期望**:
- Type: Frequency Domain
- Frequencies: f0 = 40000 Hz (= 40 kHz,单点)
- Parameter list: 长度 1

**怎么验证**:
- Study 1 → "Frequency Domain" 标签 → 看到 plist
- Solver 1 → 默认设置(Stationary / Frequency Domain),MUMPS 求解器

### 1.6 看结果 (Results)
**应该有** (如果模型是 PML 模型):
- Data Sets: Solution 1 (dset1) → 默认 2D 全场数据
- Plot Groups: 我没在 PML 模型里建 Plot Group,所以你需要**自己建**(下面 1.7 教)

**Plane baseline `step1_plane_baseline` 已经预设**:
- pg2d: 2D 压力分布
- pg_axis: 轴上曲线(r=0)
- 数据集: cpl_axis (Cut Line 2D, r=0, z=-300→200)

### 1.7 提取轴上曲线 (你必须自己做的核心验证)

**步骤** (在 COMSOL Desktop):
1. 模型加载完成后,顶部菜单: **Results → Evaluation Group → 新建 "eval_axis"**
2. 选 **"Evaluate Along Edge"** (沿边线求值)
3. **Selection**: 选 z 轴 (左击模型图上从 (0, -300) 到 (0, +200) 的一条线)
   - 在 plot 视图里点 z=0 处就是轴 — 也可以用 Box: x∈[-0.5, 0.5], y∈[-300, 200]
4. **Expression**: `abs(acpr.p_t)` (绝对声压)
5. **Unit**: `Pa`
6. 点 `Evaluate` → 看表格或生成 Plot Group 1D

**或者更简单 — 用 Cut Line 2D**:
1. Results → Data Sets → 右键 → "Cut Line 2D"
2. Name: `axis_line`
3. 线定义: 起点 (0, -300), 终点 (0, +200)
4. Apply
5. Results → Plot Group 1D → Line Graph:
   - Data: axis_line
   - Expression: `abs(acpr.p_t)`
   - Unit: Pa
6. Plot → 看曲线,记录**峰值位置 z_max 和峰值大小 p_max**
